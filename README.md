# MUTE-python-translator
**Mute Translator** — быстрый переводчик для Windows 10/11 в стиле Acrylic Blur. Выделяйте текст и переводите в один клик без браузера.  * **Удобно:** Мгновенный перевод прямо в окне. * **Стильно:** Темная тема и современный дизайн. * **Просто:** Автоустановка за секунды без прав администратора.  Сделайте работу с иностранными текстами комфортной!!
